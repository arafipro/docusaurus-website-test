- 1_install_langchain.md
- 2_setup_langsmith.md
- 3_choose_language_model.jmd
- 4_install_dependencies.md
- 5_set_environment_variables.md
- 6_instantiate_model.md
- 7_use_model.md

- 10_create_chain.md

これらのファイル名は、各手順の内容を簡潔に表しており、順番に実行しやすいように番号を振っています。
