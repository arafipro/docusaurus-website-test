import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

LangChain をインストールするには、以下のいずれかのコマンドを使用します。

<Tabs>
  <TabItem value="npm" label="npm" default>
    ```bash
    npm i langchain
    ```
  </TabItem>
  <TabItem value="yarn" label="yarn">
    ```bash
    yarn add langchain
    ```
  </TabItem>
  <TabItem value="pnpm" label="pnpm">
    ```bash
    pnpm add langchain
    ```
  </TabItem>
</Tabs>

これらのコマンドは、選択したパッケージマネージャー（npm、yarn、pnpm）を使用して、LangChain とその依存関係をプロジェクトにインストールします。
